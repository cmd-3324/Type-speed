# ChaosType — Terminal Typing Speed Test

**Randomized typing test with live scoring and performance charts.**

Built from scratch in Python. No tutorials. Pure OOP architecture.

---

## What It Does

Generates random character strings. You type them. It measures your speed.

- **Random strings** — configurable length, symbols, numbers, case
- **Live WPM & accuracy** — instant feedback after each round
- **Performance chart** — line graph of speed over rounds
- **Customizable** — change chunk size, symbols, difficulty
- **Clean CLI** — custom menu framework with box-drawing
